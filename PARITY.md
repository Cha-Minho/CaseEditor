# 기존 HTML 버전과 맞출 항목

React/PWA 전환본은 계정 저장, 폰 동기화, 배포 구조를 먼저 갖춘 버전입니다. 기존 `case-notes-organizer-api-v6.html`의 조작감과 화면을 완전히 따라오려면 아래 항목을 별도 안정화 대상으로 봅니다.

## 우선순위 높음

- 기존 HTML에 가까운 editor 밀도와 색감으로 조정
- 목차/검색 팝업의 열림/닫힘 동작 전체 확인
- `Esc`로 팝업 닫기
- 목차와 판례 드래그 선택/이동 동작 비교
- 형광펜/지우개 모드의 마우스 동작 비교
- 모바일에서 목차, 검색, editor 전환 흐름 확인

## 현재 반영됨

- `Esc`로 열린 팝업 닫기
- 계정 기반 저장
- PC/폰 동기화
- 로컬 데이터 계정 승격
- 기존 JSON 가져오기

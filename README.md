# 판례 정리함

법 공부 중 판례를 수집하고, 목차별로 정리하며, 원문과 개인 메모를 함께 관리하는 단일 HTML 앱입니다.

## 주요 기능

- 사건번호 기반 판례 추가 및 국가법령정보 API 연동
- 원하는 깊이의 목차 트리 관리
- 팝업형 목차/검색 패널
- 판례 원문, 판시사항, 판결요지, 주요 문구, 내 요약, 의견별 메모 작성
- 참고자료/내 정리 영역 너비 조절
- 편집 영역 전체 형광펜/지우개 도구
- 브라우저 저장 및 선택한 JSON 파일 수동/자동 저장

## 사용 방법

`case-notes-organizer-api-v6.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

API 자동 불러오기를 사용하려면 좌측 툴바의 `API 키(OC)` 칸에 국가법령정보 공동활용 API 인증값을 입력하세요.
